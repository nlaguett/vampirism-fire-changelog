# Vampirism Fire 8.0g

---

## Bug Fixes & QoL Changes

---

## Main

- The `-oust` command now correctly ousts **ally buildings** from your base:
  - Slayer Tavern
  - Research Center
  - Houses and Tents
- Fixed the **Gold Mine cancel bug** that would remove income when canceling a Gold Mine
- Reworked the code that displays the **Fang Harvester trained** message
- Reworked the code for first **Citadel / Command Center / Base of Operations** messages
  - All first train / construct popups now display **“A Player”** instead of **“A Human”**
- Engineers no longer take **2500 damage** from **Rain of Blood**
- Fixed a bug that would kill the **Research Center** when picking up equipment
- Updated clan tags

---

## Architect

- Fixed selection scale for the following buildings:
  - **House / Lumber House / Defo House**
  - **Base of Operations**
  - **Mini Gold Mine**
  - **Aqua Outpost**

---

## Orc

- Fixed selection scale for the following buildings:
  - **Orc’s Vault**
  - **Great Hall / Stronghold / Fortress**
- Fixed **Fang Blade** so it now correctly applies the buff to newly trained **Fang Harvesters**
- **Sapphire Wall**
  - Armor bonus increased from **55 → 65**
- **Emerald Wall**
  - Armor bonus increased from **80 → 85**

---

## Vampires

- **Curse of Doom**
  - Can now be cast on **Architect Furbolg / Super Tower Builder**
  - Can now be cast on **Orc Furbolg / Super Tower Builder**
- **Assassin**
  - Can now attack **Architect Harvesters**
- **Reaper**
  - Removed auto-selection (was causing issues)
  - Reaper priority remains higher than Vampires, making it easier to select

## Balance Changes

---

  - Damage stack increased from **250 → 750**
- **Metal Amethyst Wall**
  - No longer stacks **Punisher** stats
- **Sapphire Wall**
  - No longer stacks **Punisher** stats
- **Lumber Mill**

    - Level 2: **300**

    - Level 4: **2500**
- **Barricade Builder**
  - Removed
  - Architect now uses **Boulder Tower** to block Vampires during early rushes
- **Boulder Tower**
  - HP increased from **5000 → 6500**
